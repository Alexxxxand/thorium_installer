# Installing

### With "apt"

Installing <code>git</code>

```bash
sudo apt install git
```

### With "pacman"

Installing <code>git</code>

```bash
sudo pacman -S git
```

### With "dnf"

Installing <code>git</code>

```bash
sudo dnf install git
```

### Cloning repository

```bash
git clone https://github.com/Alexxxxand/thorium_installer
```

### Moving to directory

```bash
cd thorium_installer
```

### Installing 

```bash
./install.sh
```
